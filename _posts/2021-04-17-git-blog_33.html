각 항목에 대한 매뉴얼은

https://mmistakes.github.io/minimal-mistakes/docs/quick-start-guide/

여기서 확인


변경된 내용 로컬호스트에 반영하는방법

_config.yml 파일 수정후에 변경된 내용을 반영하기위해 다음 명령어를 cmd에서 실행함

cd [repository 경로]

bundle exec jekyll serve


localhost:400 에 들어가면 변경된 내용 확인가능.


깃허브에 업로드하는 방법

cd [repository 경로]

git add .

git commit -m "[업로드 내용 등 메세지]"

git push
