깃허브 블로그 만드는법

Step 1. 깃허브 Repository 하나 새로 만듬.

만들때 Repository 이름을 "[본인 ID].github.io" 형식으로 만듬, Readme Install 에 체크함


Step 2. 테마 클론 다운로드함. 압축풀음

http://jekyllthemes.org/ 

에서 테마선택가능함


Step 3. cmd 창을 켜서 Repository 를 저장할 디렉토리로 이동함

(내 경우엔 D 드라이브에 저장했음.)


*cmd 창에서 D 드라이브로 이동하는 방법.

D:



CD 명령어로 저장할 repository 로 이동함.



Step 4. Clone url 복사하기








해당 Repository 메인화면에서 Code 버튼을 눌러서 url 클립보드에 복사함




Step 5. git clone [원격 저장소 url] 명령어로, repository 내용을 폴더에 다운로드 복제함

cmd 창에서 git clone 명령어를 사용해서 repository 내용을 폴더에 다운로드함 (복제)


git clone [아까 복사한 url]







성공하면 폴더가 하나 생김







Step 6. 테마 적용하기

Step 2에서 다운로드한 테마를 압축풀고, 아까 clone 다운로드한 repository 폴더에 내용물 복사함













CD 명령어로 아까 다운로드한 repository 로 이동

cd ./[새로생긴 폴더 경로]




다음 명령어를 cmd에서 실행

gem install bundler

bundler install





Repository 폴더내의 Config yml 파일을 메모장 등 에디터로 열어서 url 수정함





url 은 [내 url 주소] , base url 은 "" 으로 값을 변경해주어야함
#앞에다가 url 주소 입력







Step 7. 테마 적용 확인하기

다음과 같은 명령어를 cmd 에서 실행함
git add .
git commit -m "Theme changed"
git push






*git commit -m "Theme changed" 할때 git please tell me who you are 메세지 해결방법



git config --global user.eamil <"사용할 이메일">

git config --global user.name  <"사용할 이름">

커맨드로 해결





*git push 시에 logon failed use ctrl+c to cancel basic credential prompt

에러 뜨는데 그냥 cmd 창에서 아이디, 패스워드 치고 그냥 진행함





Step 8. 실제 블로그에 접속해보기
Step 1에서 [본인 ID].github.io 로 저장소 이름 설정했는데
인터넷 탐색기 주소창 에다가 [본인 ID].github.io 입력후 들어가면 테마 변경된 것 확인가능
