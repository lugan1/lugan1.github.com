 jekyll 에 이미지 업로드 하는방법

![이미지이름](이미지 주소)

이미지 이름은 이미지 엑박뜰때 이미지 이름이 표시됨.

이미지 주소는 로컬경로 or 이미지 링크주소 둘중 하나로 적으면됨


로컬경로는 [repository]/

부터 시작함. 주로 assets 폴더에다가 post용 이미지 폴더 새로 만들고 거기다가 저장하는듯



ex) [repository]/assets/image/posts_image 에 create_post.png 이미지 파일이 있을시에 

![testImage](/assets/image/posts_image/create_post.png)


*주의. 이미지 파일 이름이 한글이면 jekyll 로컬 구동시에 오류남




이미지 업로드 방법은 총 4가지가 있음.


1. repository 폴더 내에 이미지 파일채로 git push 해서 업로드하는 방법.

- 이경우엔 이미지 url 에다가 파일 경로를 적음.

- repository 무료 사용 용량이 1GB 제한이라서 1GB 넘어갈시에 방법을 찾아야함.


2. imgur 사이트에 이미지 업로드 후에 이미지 링크 거는방법.

- 이미지 한번 업로드하면 삭제 절차가 까다로움.


3. 구글 드라이브에 업로드 후에 이미지 링크 거는방법.

- 무료 용량 15GB 제한


4. Github issue 기능 이용해서 업로드
