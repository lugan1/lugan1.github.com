 Ruby, Jekyll 사용해서 로컬에서 커스터 마이징하고, 변경된 화면 확인가능함

-Jekyll bundler 툴 사용

-bundle exec jekyll serve 명령어를 이용해서 실시간으로 로컬에서 수정된 화면 확인가능

(localhost:4000)



Step 1. Ruby 설치

Ruby 설치 방법

https://rubyinstaller.org/downloads/

에서 설치



Step 2. jekyll 설치

cmd 창 열고 이전 포스트에서 Repository 폴더로 이동한 후에 다음 명령어 실행

gem install jekyll bundler


Step 3. 로컬 호스트에서 내 블로그 확인하기

Repository 폴더 내에서 cmd 다음 명령어 실행 

(중요. cmd 에서 Repository 폴더로 이동한다음에 실행해야됨)

bundle exec jekyll serve



*Please append `--trace` to the `serve` command 

cannot load such file webrick 에러 해결방법

(weblick 을 로드하지 못함)




에러나는 이유
-ruby 3.0.0 부터 weblick 이 기본으로 포함된 gem 에서 빠졌기 때문에 따로 weblick 추가해야됨




다음 명령어 실행

bundle add weblck

(weblick 추가) 




다시 cmd 창에서

bundle exec jekyll serve

명령어 실행하면 서버가 구동되는것을 확인가능함





인터넷 주소창에 localhost:4000 치고 이동하면 블로그 화면 나오는것을 확인가능
